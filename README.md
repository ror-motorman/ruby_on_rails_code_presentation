# Feedback Form

### Used technologies

  * Ruby 2.3.0
  * Rails 4.2.6
  * MySQL
  * jQuery
  * jQuery validation (Client Side Validation)
  * Coffeescript
  * Bootstrap 3
  * CSS/SCSS/SASS
  * HTML/HAML/ERB
  * i18n
  * Simple Form
  * CarrierWave
  * MiniTest

### Install

##### 1. Create `database.yml`. Example: `/config/database.yml.example`.

##### 2. Run Bundle:
```console
bundle install
```   
##### 3. Create DB:
```console
rake db:create
``` 
##### 4. Run migrations:
```console
rake db:migrate
```  

### Testing
```console
rake test
```
